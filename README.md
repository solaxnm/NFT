# NFT Learn by SolanaM - Complete NFT Education Hub

A comprehensive educational platform designed to master Non-Fungible Tokens from complete beginner to advanced developer. Built with Nuxt 3 and modern web technologies.

## 🚀 Platform Overview

NFT Learn by SolanaM is a cutting-edge educational hub that provides structured learning paths for understanding, creating, and developing with NFTs. The platform covers everything from basic blockchain concepts to advanced smart contract development and NFT infrastructure.

### Key Features

- **📚 Structured Learning Paths**: Four comprehensive courses from beginner to expert level

- **🌐 Modern Tech Stack**: Built with Nuxt 3, Tailwind CSS, and TypeScript
- **📱 Responsive Design**: Optimized for all devices and screen sizes
- **⚡ Performance Optimized**: Fast loading and smooth user experience

## 🛠️ Technical Stack

- **Framework**: Nuxt 3 with TypeScript
- **Styling**: Tailwind CSS with custom components
- **Icons**: Heroicons via @nuxt/icon
- **Content Management**: @nuxt/content for markdown-based content
- **Build Tool**: Vite (via Nuxt 3)
- **Package Manager**: pnpm (recommended)

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm/yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/solaxnm/nft-education-hub.git
cd nft-education-hub

# Install dependencies
pnpm install
# or
npm install
# or
yarn install

# Start development server
pnpm dev
# or
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:3000`
